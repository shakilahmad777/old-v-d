1. What is the difference between map, filter, foreach ?

map(), filter(), and forEach() are all higher-order functions in JavaScript that operate on arrays, but they have different purposes and return values.

map() transforms each element in an array and returns a new array, filter() returns a new array containing only the elements that pass a test, forEach() executes a function for each element in the array without modifying the array or returning anything.

2. What is the difference between local storage, session storage, cookies ?

LocalStorage, SessionStorage, and Cookies are all used for client-side storage in web browsers, but they have different characteristics and use cases.

LocalStorage and SessionStorage are used for client-side storage in the browser, with LocalStorage being permanent and available across all browser windows and tabs, and SessionStorage being temporary and available only within the same browser window or tab. Cookies are also used for client-side storage but are typically used for smaller amounts of data that need to be remembered across multiple requests and have an expiration date and size limit.

1. What is the Event Loop?

An event loop is a programming construct used in event-driven programming, which is a programming paradigm where the flow of the program is determined by external events such as user actions or system events rather than being driven by the program's code.
