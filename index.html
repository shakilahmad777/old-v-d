Difference Between Flex and Grid?

Flexbox and Grid are both layout systems in CSS that allow you to create flexible, responsive designs for web pages.

Flexbox is a one-dimensional layout system, meaning that it deals primarily with layout in a single dimension (either rows or columns).

Grid, on the other hand, is a two-dimensional layout system that allows you to create grid-based layouts. It is best suited for creating more complex layouts that involve multiple rows and columns.

What is Media quaries?

Media queries are a feature of CSS that allow you to apply different styles to a document based on the characteristics of the device displaying it.
