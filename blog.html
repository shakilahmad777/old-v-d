What is Dom ?

The Document Object Model (DOM) is a programming interface for web documents. and cross-platform and language-independent interface. The DOM represents a document with a logical tree.

What is Event Bubbling

Event Bubbling is a concept in the DOM (Document Object Model). It happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.

What is Calback Function

A callback is a function passed as an argument to another function. This technique allows a function to call another function. A callback function can run after another function has finished.
